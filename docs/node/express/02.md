# 二、基本使用

1. 在项目中安装 `express` 

   ```javascript
   npm i express
   ```

2. 创建基本的web服务器

   ```javascript
   const express = require('express')
   
   const app = express()
   
   app.listen(8080, () => {
       console.log('listening on http://localhost:8080')
   })
   ```

## 2.1 监听请求

**GET**

```javascript
app.get('请求URL', (req, res) => {
    // 向客户端发送数据
    res.send({username: 'jun233s'})
})
```

**POST**

```javascript
app.post('请求URL', (req, res) => {
    // 向客户端发送数据
    res.send('请求成功')
})
```
